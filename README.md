\# 🍽️ Food Calorie Detection using Deep Learning



\## 📌 Project Description



Food Calorie Detection is a machine learning-based web application that identifies food items from images and predicts their approximate calorie content.



The application uses a deep learning model to classify food images and provides calorie information with prediction confidence through a Flask web interface.



\---



\## ✨ Features



\- Upload food images for detection

\- Deep learning based food classification

\- Predicts food name and calorie value

\- Displays prediction confidence percentage

\- Flask web application interface

\- Simple and user-friendly design

\## 🚀 How to Run



\### Step 1: Clone the repository



git clone https://github.com/monikasree-19/Food-Calorie-Detection.git



\### Step 2: Open project folder



cd Food-Calorie-Detection



\### Step 3: Install required libraries



pip install -r requirements.txt



\### Step 4: Run the application



python app.py



\### Step 5: Open in browser



http://127.0.0.1:5000/



Upload a food image and check the predicted food name, calorie value, and confidence percentage.
